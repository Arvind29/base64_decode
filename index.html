Base64 Decoder

Instantly decode Base64 strings to readable text

What is Base64?

Base64 is a method to encode binary data into ASCII characters. It's commonly used for data URLs, email attachments, and basic obfuscation.

How to use

  1. Paste your Base64 string
  2. Click "Decode"
  3. Copy or edit the result

Example

Try this sample: TWFuIGlzIGRpc3Rpbmd1aXNoZWQ= (decodes to "Man is distinguished")
